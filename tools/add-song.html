How to add a song:
  1. Paste or drop a Spotify track link above
  2. Confirm the date (defaults to today)
  3. Copy the YAML entry
  4. Open songs.yml on GitHub
  5. Paste the entry at the very top of the file (before all other entries)
  6. Commit the change — done!
